# mattergen-x

A research-grade AI material discovery and visualization platform.

## Monorepo Structure

- [frontend/](./frontend/): Next.js application for material visualization and user interface.
- [backend/](./backend/): FastAPI application providing AI inference and data management services.
- [training/](./training/): ML training scripts and experiment configurations.
- [models/](./models/): Storage for saved model weights and architecture definitions.
- [data/](./data/): Raw and processed datasets for training and validation.
- [shared/](./shared/): Shared schemas, types, and utility functions used by both frontend and backend.
- [scripts/](./scripts/): Helper scripts for development, deployment, and data processing.
- [docs/](./docs/): Technical documentation, architecture diagrams, and research notes.
